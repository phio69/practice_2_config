Инструмент визуализации графа зависимостей для менеджера пакетов (NuGet)

Реализация задания по дисциплине «Конфигурационное управление» (вариант №21).
Инструмент анализирует зависимости пакетов NuGet и строит граф зависимостей с поддержкой транзитивности, обнаружения циклов и обратных зависимостей.

Возможности:
Получение прямых и транзитивных зависимостей пакета NuGet.
Поддержка тестового режима с графом из букв (A -> B -> C).
Обнаружение циклических зависимостей.
Вывод обратных зависимостей (пакеты, которые зависят от указанного).
Генерация Mermaid-кода для визуализации графа.
Сохранение графа в файл .mmd для дальнейшего преобразования в SVG.

Параметры командной строки: 
--package - Имя анализируемого пакета NuGet (например,Newtonsoft.Json)
--repo-url - URL репозитория NuGet или путь к JSON-файлу в тестовом режиме
--version - Версия пакета
--output - Имя файла для сохранения Mermaid-кода
--test-mode - Активировать тестовый режим с графом из букв
